# 幾何学戦争 (The Battle Geometry)

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![Pygame](https://img.shields.io/badge/Library-Pygame-green.svg)

幾何学図形を召喚して敵の城を落とす、2ライン制のタワーディフェンスゲームです。
味方キャラクターの描画に画像ファイルを使わず、プログラム（`generator.py`）によって図形をリアルタイムに生成しているのが特徴です。

![Game Screen](image/sakusei%20haikei.jpg)

## ゲーム概要
* **ジャンル**: タワーディフェンス
* **プレイ人数**: 1人
* **ステージ数**: 全3ステージ + チーム編成機能

プレイヤーは時間経過で貯まる「資金」を消費して幾何学ユニットを召喚し、自軍の城（右側）を守りつつ、敵軍の城（左側）を破壊することを目指します。

## 操作方法

| 操作 | アクション |
| :--- | :--- |
| **マウス左クリック** | ユニットの召喚 / ステージ選択 / チーム編成 |
| **Space キー** | **出撃レーン（上・下）の切り替え** |
| **Esc キー** | ゲーム終了 |

## ゲームシステム

### 1. 2ライン戦略
フィールドは上下2つのレーンに分かれています。
`Space` キーを押すと、ユニットを出撃させるレーン（[上] または [下]）が切り替わります。敵の攻め手薄なレーンを狙ったり、攻められているレーンを防衛したりする判断が重要です。

### 2. チーム編成
全12種類のユニットから、最大10体を選んでデッキを組むことができます。
「チーム編成」画面で、コストや性能の異なるユニットを組み合わせて戦略を練ってください。

### 3. ステージ構成
* **ステージ1**: 初級。基本的な敵が出現します。
* **ステージ2**: 中級。敵の数が増え、城の耐久力が上がります。
* **ステージ3**: 上級。強力なボス（スターボス、ヘキサボス、立体魔王）が登場します。

## ユニット紹介 (Units)
各ユニットは独自のステータスと役割を持っています。
各ユニットのステータスは自由に変更可能です。

| 名前 | 形状 | コスト | 特徴 |
| :--- | :--- | :--- | :--- |
| **トライアングル** | 正三角形 | 50 | 最速・低コスト。特攻向けだが打たれ弱い。 |
| **スクエア** | 正方形 | 200 | バランス型。攻守の要。 |
| **ペンタゴン** | 正五角形 | 400 | タンク。移動は遅いが耐久力が高い。 |
| **ヘキサゴン** | 正六角形 | 500 | 範囲攻撃が得意な中衛ユニット。 |
| **サークル** | 円 | 250 | バウンサー。特殊な挙動で撹乱する。 |
| **スター** | 星型 | 600 | 高火力アタッカー。 |
| **クロス** | 十字 | 300 | 支援型。射程が長く、後方から援護する。 |
| **ロンバス** | ひし形 | 450 | 貫通攻撃持ち。 |
| **クレセント** | 三日月 | 350 | ブーメランのような特殊軌道で攻撃。 |
| **トラペゾイド** | 台形 | 900 | 重戦車。超鈍足だが圧倒的なHPを持つ。 |
| **ヘキサグラム** | 六芒星 | 800 | 魔法使い。強力な遠距離攻撃を行う。 |
| **フラクタル** | 雪の結晶 | 2000 | **最強ユニット**。コストは高いが攻守ともに最強。 |

## インストールと実行

必要なライブラリは `pygame` のみです。

```bash
# リポジトリのクローン
git clone [https://github.com/TCRita/The-battle-geometry.git](https://github.com/TCRita/The-battle-geometry.git)
cd The-battle-geometry

# ライブラリのインストール
pip install pygame

# ゲームの実行
python main.py
